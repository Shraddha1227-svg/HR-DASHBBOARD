# HR-DASHBBOARD
This repository contains an interactive HR Analytics Dashboard built using Microsoft Power BI to analyze workforce data and provide actionable insights for HR decision-making
Project Overview

The dashboard helps organizations monitor key HR metrics such as employee demographics, attrition, department-wise performance, job roles, salary distribution, and workforce trends through interactive visualizations.

Features
Employee headcount analysis
Attrition rate and turnover insights
Department-wise employee distribution
Gender and age diversity analysis
Job role and education breakdown
Salary and compensation analysis
Interactive filters and slicers for dynamic reporting
Clean and user-friendly dashboard design
Tools & Technologies
Microsoft Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Excel Dataset
Key Insights
Identifies departments with the highest attrition.
Analyzes workforce distribution across age groups and genders.
Tracks employee count by education level and job role.
Provides salary and experience-based insights for better HR planning.

This project demonstrates how Power BI can transform raw HR data into meaningful business insights, enabling data-driven workforce planning and strategic decision-making.
